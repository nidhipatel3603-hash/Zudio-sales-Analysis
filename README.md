# Zudio-sales-Analysis
This project analyzes sales data of a retail brand (Zudio) using SQL, Excel, and Power BI.
The goal is to extract insights like revenue trends, top-selling products, and category performance.
SQL (Data Extraction & Queries)
Excel (Data Cleaning & Preparation)
Power BI (Dashboard & Visualization)
Identified top-performing product categories
Analyzed monthly sales trends
Compared revenue vs profit
Found top-selling products

Conclusion
This project demonstrates end-to-end data analysis skills from raw data to interactive dashboard.
